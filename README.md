1)Название проекта - DиноZаврик

2)project by Denis K.

3)Описание проекта:
Пародия на популярную игру в Chrome с добавлением авторских фич и элементо; базы данных с результатами игроков.
Игрок будет управлять динозавриком, который перепрыгивает различные препятствия/движущиеся объекты

4)Техническое задание/реализация:

4.1)Реализация классов и разбиение их по отдельным файлам

class Dyno: класс игрока, функции update и  update_texture меняют модельку игрока и его скорость

class Pterodactyl, Cactus: классы препятствий/коллизий

class MyGame - основной класс игры с базовыми(setup, on_draw, on_update, on_key_press) и дополнительными функциями(make_cactus, make_pterodactyl)

4.2)Выполнение критериев проекта:

-Добавление стартового и финального окна с подсчетом результатов(self.start_game, self,end_game, self.count)
-Реализация "фонового движения"(3 спрайта попеременно заменяют друг друг при движении игрока; self.background)
-Добавление камеры и ее настройка
-Добавление звука и работы с частицами GUI
-Реализация работы с БД(players_data.sqlite3; player-count)
-Реализация смены моделек игрока при движении

5)Библиотеки, использованные в проекте:
arcade, random, pyglet.graphics
