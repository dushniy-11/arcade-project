DиноZаврик
Описание проекта - пародия на популярную игру в Chrome с добавлением авторских фич и элементов и базы данных с результатами игроков.
Игрок будет управлять динозавриком, который перепрыгивает различные препятствия/ движущиеся объекты

Техническое задание:

-создание классов игры(MyGame), персонажа(Dyno), кактусов(Cactus), птеродактелей(Pterodactyl) , разбиение их по файлам, добавление им моделек

-создание и настройка игры, списка спрайтов "противников", который будет рандомизироваться при каждой новой игре

-реализация базовых функций класса игры(setup, on_draw, on_update, on_key_press и т.д.)

-реализация главного меню(стартового окна), финального окна, подсчета результатов

-добавление звука(фоновая музыка, прыжок, проигрыш и др.)

-добавление авторских фич

-реализовать БД с результатами игроков(например: кол-во прохождений игры)

-протестировать код, исправить ошибки
