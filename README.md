DиноZаврик
Описание проекта - пародия на популярную игру в Chrome с добавлением авторских фич и элементов, а также наличие концовки, 2 уровня, база данных с результатами игроков.
Игрок будет управлять динозавриком, который перепрыгивает различные препятствия/ движущиеся объекты в 1 уровне(крч оригинал)
Во 2 уровне будут сделаны различные механики боя и движения, например более высокий прыжок, файрбол, щит и т.д.

Техническое задание:

-создание классов игры(Game), персонажа(Dyno), кактусов(Cactus), птеродактелей(Pterodactyl) , разбиение их по файлам, добавление им моделек

-создание первого(базового) уровня, списка спрайтов "противников", который будет рандомизироваться при каждой новой игре

-реализация базовых функций класса игры(setup, on_draw, on_update, on_key_press и т.д.)

-создание 2 уровня, где будут добавлены новые фичи, соответственно их реализация в отдельных функциях( и добавление им логики для взаимодействия с объектом игрока(смена его модельки, высоты прыжка, добавление неуязвимости и т.д)

-реализация главного меню(стартового окна), финального окна, подсчета результатов

-добавление звука(фоновая музыка, прыжок, проигрыш и др.)

-реализовать БД с результатами игроков(например: кол-во прохождений игры)

-протестировать код, исправить ошибки
